# Product Gallery with Light/Dark Mode

A responsive product showcase gallery with theme toggle and smooth animations.

## Features
- 3 product cards with hover effects
- Light/Dark mode toggle with localStorage persistence
- Fade-in animations with Intersection Observer
- Responsive design
- Interactive notifications and ripple effects

## Technologies
- HTML5
- CSS3 (CSS Variables, Grid, Flexbox)
- JavaScript (ES6+)
